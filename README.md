# Omni 🔍

> Misura la presenza online di un'attività locale e suggerisce come migliorarla.

## Cosa fa

Omni analizza la visibilità online di un'attività (ristorante, negozio, studio professionale, ecc.) e produce:

- uno **score di visibilità** da 0 a 100
- un **report dettagliato** con i punti di forza e debolezza
- una lista di **suggerimenti prioritizzati** per migliorare la presenza online

## Come funziona

1. Inserisci il nome dell'attività e la città
2. Omni raccoglie dati da Google Search e Google Maps
3. Analizza il posizionamento, le recensioni e la coerenza delle informazioni
4. Genera un report con uno score e suggerimenti concreti

## Tecnologie usate

- Python 3.x
- Google Places API
- Google Custom Search API
